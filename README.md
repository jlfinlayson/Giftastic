# Giftastic!

Giftastic is a web application that allows users to view gifs of their favorite artists. The application features preset options as well as the option to search for the users artist of choice.<br>

 
 ### Technologies Used

* HTML
* CSS
* Bootstrap
* Javascript
* jQuery
* Giphy API
